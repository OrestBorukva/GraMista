## Як запустити локально

Див. [README.md](README.md) — розділ «Запуск (self-host)». Коротко:
`npm install` → `docker compose up -d` → `npm run db:seed` → `npm run dev`.

Перед PR: `npm run typecheck && npm run typecheck:test && npm test` — все зелене.
