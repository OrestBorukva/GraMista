# GraMista 🗺️

> Gamified charity-donation tool for Ukrainian streamers: viewers donate via
> monobank jar and name a Ukrainian city in the comment — the city scores
> points on a live map with OBS overlays. Free for noncommercial use.

Гейміфікація донатів через змагання міст України. Глядач донатить на банку
monobank і пише в коментарі своє місто → місто отримує бали → живий топ міст,
інтерактивна мапа України (29 242 населені пункти) і прозорі оверлеї для OBS.

**Можливості:**

- 🗺️ Жива мапа: місто «засвічується» від першого донату, спалахує від нових
- 🏆 Топ міст, стрічка донатів, гонка міст, скарбнички-збори як сезони
- 🎥 6 OBS-оверлеїв (алерт, топ, стрічка, мапа тощо) — конструктор з прев'ю
- 📺 Публічна сторінка стрімера для глядачів: мапа, топ, QR на банку
- 🤝 Спільна мапа України з усіх учасників (опт-ін)
- 🔒 Токен monobank **не зберігається**: разове використання при підключенні
  банки, далі працює лише вебхук

## Запуск (self-host)

Потрібні: Node.js 20+, Docker (для Postgres).

```bash
npm install
docker compose up -d        # Postgres
npm run db:seed             # стартові дані (одноразово)
npm run db:import           # повний датасет НП (одноразово; спершу поклади
                            # відкриті дані в data/sources/ — URL у шапці
                            # prisma/import-settlements.ts)
npm run dev                 # http://localhost:3000
```

Прод: `docker-compose.prod.yml` (web + Postgres + Caddy з авто-HTTPS);
змінні — за шаблоном `.env.prod.example`. Підключення банки monobank —
у налаштуваннях профілю (токен використовується разово й знищується).

Тести: `npm test` · типи: `npm run typecheck`.
